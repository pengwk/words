# 优化程序

## 词频统计

三种方法①直接使用`dict`②使用`defaultdict`③使用`Counter`

ps:`int()`函数默认返回0

①
```python
text = "I'm a hand some boy!"

frequency = {}

for word in text.split():
    if word not in frequency:
        frequency[word] = 1
    else:
        frequency[word] += 1
```

②defaultdict
```python
import collections

frequency = collections.defaultdict(int)

text = "I'm a hand some boy!"

for word in text.split():
    frequency[word] += 1

```

③Counter
```python
import collections

text = "I'm a hand some boy!"
frequency = collections.Counter(text.split())
```
